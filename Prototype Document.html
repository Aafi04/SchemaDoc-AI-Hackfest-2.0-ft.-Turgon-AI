System Architecture

Data Ingestion SQLAlchemy Core
Multi-dialect enterprise connections — SQLite, PostgreSQL, Snowflake — via dialect-agnostic inspect methods. Structured metadata extraction without ORM overhead.
↓
Orchestration LangGraph State Machine
Cyclic state machine managing bidirectional flow between deterministic SQL extraction and probabilistic AI enrichment. Enables controlled retry loops and branching validation paths.
↓
Enrichment Engine Gemini 2.5 Flash · LangChain ReAct
ReAct pattern agent resolves ambiguous column taxonomy via forensic log searches through tool binding. Infers business logic and semantic meaning from raw schema metadata.
↓
Validation Gate Deterministic Rule Engine
Compares LLM output schemas against SQL skeleton. Hard blocks on data dropping or hallucination. Triggers mandatory retry loops — LLM output is never passed downstream unverified.
↓
Presentation Streamlit · streamlit-agraph
Streamlit runtime hosts the data dictionary interface, ER visualization via (streamlit-agraph) force-directed graphs, and the LLM chat interface for natural language SQL generation.
